# Embedded 타입
- 하이버네이트에서 임베디드 타입을 컴포넌트 타입이라 함 ⚠️

### 이유
- 코드의 재사용
- 높은 응집도

### 사용법
- 기본 생성자가 필수
- @Embedded
    - 값 타입을 사용하는 곳에 표시
- @Embeddable
    - 값 타입을 정의하는 곳에 표시
```java
@Embeddable
@Getter
public class Address {
    private String city;
    private String street;
    private String zipcode;
    }

    @Builder
    public Address(String city,String street,String zipcode){
        this.city=city;
        this.street=street;
        this.zipcode=zipcode;
    }
}
```
```java
@Entity
@Getter 
public class Member {
    @Id
    @GeneratedValue
    @Column(name = "member_id")
    private Long id;

    private String name;

    @Embedded
    private Address address;

}
```
