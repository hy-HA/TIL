## 2. LIMIT [offset,] row_count
- 테이블에 들어있는 데이터를 페이지 별로 가져오려면 select문의 limit 구문 사용
- offset
    - 처음으로부터 얼마나 떨어져 있는가
    - 10 offset : 처음으로부터 10만큼 떨어져 있음 = 11번째
    - 1페이지 = offset 0 = (page-1)*10
    - 2페이지 = offset 10 = (page-1)*10
    - 3페이지 = offset 20 = (page-1)*10

- row_count
    - 읽어올 row 수
    - 10 row_count = 10줄 가져오기
